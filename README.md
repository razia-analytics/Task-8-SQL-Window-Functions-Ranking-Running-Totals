# Global Superstore SQL Analysis

SQL-based analysis of the Global Superstore dataset to understand sales trends, customer behavior, and product performance using aggregations and window functions.

## Key Analysis

* Customer and regional sales ranking
* Month-over-month sales growth
* Running total sales over time
* Top-performing products by category

## Tools

* MySQL Workbench
* SQL

## Notes

* Date fields converted using `STR_TO_DATE()` where required.
* Window functions used for ranking and trend analysis.
